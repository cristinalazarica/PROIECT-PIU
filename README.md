#  Temă: Evidența comenzilor și a stocului de flori

### Aplicația permite adăugarea, editarea și ștergerea florilor și clienților. Fiecare floare are nume, preț și stoc disponibil.

### Se pot înregistra comenzi care conțin clientul, produsele, data livrării și istoricul comenzilor, inclusiv numărul de comenzi active ale unui client. La plasarea comenzii se verifică stocul și, dacă acesta este insuficient, se afișează un mesaj de avertizare.

### Aplicația poate afișa cele mai vândute flori, vânzările pe o perioadă și comenzile dintr-o anumită zi.
